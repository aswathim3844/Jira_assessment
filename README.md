# User Authentication System (MVP)

## 📌 Project Overview
This project is a sample **User Authentication application** featuring:  

- **Login Page**  
- **Signup Page**  
- **User Dashboard**  

Developed as part of a team assessment to demonstrate **Jira-GitHub Integration Workflow**, agile task management, and AI-assisted rapid development.

---

## 🛠 Tech Stack
- **Frontend:** HTML  
- **Backend:** PHP  
- **Framework:** Laravel  
- **Version Control:** GitHub  
- **Project Management:** Jira

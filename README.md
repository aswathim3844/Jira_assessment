# User Authentication System (MVP)

## 📌 Project Overview
This project is a sample User Authentication application featuring a **Login Page**, **Signup Page**, and a **User Dashboard**. This was developed as part of a team assessment to demonstrate proficiency in the **Jira-GitHub Integration Workflow**, agile task management, and "Vibe Coding" (AI-assisted rapid development).
